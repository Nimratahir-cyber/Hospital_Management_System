# Hospital Management System

A simple console-based Hospital Management System built using C++. This project manages patients, doctors, rooms, and billing using file 
handling.

# Features

User registration and login system

Add, view, and search:

Patients

Doctors

Rooms

Room availability management

Bill generation based on stay duration and room cost

Data stored using text files

# Files Used

patient.txt → stores patient data

doctor.txt → stores doctor data

room.txt → stores room details

thisfile.txt → stores login credentials

# How to Run

Compile the code using a C++ compiler

Run the program

Register a new user

Login to access the system
